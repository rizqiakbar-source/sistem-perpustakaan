# 📚 Sistem Perpustakaan Digital

**Politeknik Negeri Lampung**  
Jurusan Teknologi Informasi - Program Studi Manajemen Informatika

## Deskripsi
Aplikasi manajemen perpustakaan berbasis web yang dikembangkan menggunakan PHP native dan MySQL. Fitur meliputi CRUD data buku, anggota, peminjaman, pengembalian, laporan, e-book, dan wishlist.

## Fitur
- ✅ Login Admin & Anggota
- ✅ Dashboard dengan Grafik
- ✅ Manajemen Buku, Anggota, Peminjaman, Pengembalian
- ✅ Laporan Transaksi
- ✅ E-Book Collection
- ✅ Wishlist
- ✅ Scan QR Code
- ✅ Dark Mode
- ✅ Responsif Mobile

## Tech Stack
- **Backend:** PHP Native
- **Database:** MySQL
- **Frontend:** HTML, CSS, Bootstrap 5
- **Library:** Chart.js, Cropper.js, html5-qrcode
